# Project AI Rules — 1-reasoning-baselines

## Scope

Baseline comparison of CoT vs Self-Consistency on public reasoning tasks.

## Do

- Classify before writing: attempt → `0-Project/`, method → `1-Docs/`, data note → `2-Data/`, paper → `3-Paper_Survey/`.
- Update `_ai/task_plan.md` at the start of a round and `_ai/progress.md` after each result.
- Keep only aggregate metrics in shared/board text; freeze benchmark versions before running.
- Update the board after any status / blocker / next-step / attempt change (markdown first, board second).

## Don't

- Don't paste raw dataset rows or full model traces into shared files or the board.
- Don't claim an improvement without a frozen benchmark and a recorded metric.
