# AI_RULES

Rules for AI collaboration inside this project. Before a new task, read `README.md`, `PROJECT_GUIDELINES.md`, and the relevant folder README.

## Project context rules

- Decide which core folder a task belongs to first, then read that README.
- For experiments, use the fields in `0-Project/experiment-template.md`.
- For data, use `2-Data/dataset-card-template.md` to track source, schema, and limits.
- For papers, use `3-Paper_Survey/paper-note-template.md` and `evidence-matrix.md`.

## Research rules

- Mark uncertain conclusions as assumption / inference / to-verify.
- Do not treat a paper abstract as a project conclusion; state its relation to this project.
- Record reproducible commands, input versions, and output locations.

## Output rules

- Prefer an actionable next step when giving conclusions.
- After a complex task, sync `_ai/progress.md`, then update `PROJECT_BOARD.html` per the sync scale.
