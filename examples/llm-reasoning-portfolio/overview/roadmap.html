Phase 0 · baselines

Portfolio board (看板) for a small set of public LLM-reasoning projects. This is a roadmap and navigation map — each project keeps its own board. Sources: overview/README.md · register · operating rules.

Current focus: freeze a shared benchmark set and land the CoT vs Self-Consistency baseline; keep search and reflection projects in literature-framing.

Roadmap

Phase 0 — BaselinesFreeze a public benchmark subset; establish CoT vs Self-Consistency. Owner project: 1-reasoning-baselines.
Phase 1 — SearchAdd inference-time search (Tree/Graph of Thoughts) on a small task, compared to the frozen baseline.
Phase 2 — ReflectionAdd self-critique/revision loops (Reflexion, Self-Refine) with a matched no-reflection control.

How they relate: baselines fixes the benchmark and metric; search and reflection reuse that frozen set so results stay comparable across projects.
