# project_overview — _PROJECT_TEMPLATE

_Copy this template to `reasoning-project/<name>/` and replace every field below._

- **Question:** what does this project answer?
- **Scope:** in / out of scope.
- **Owner:** name or temporary contact (TBD until assigned).
- **Priority:** low / medium / high / TBD.
- **Status:** template (change to preparing / active / needs review / paused / done with evidence).
- **Next action:** the single next concrete step.
- **Touches private data?** yes/no — if yes, record the boundary in `2-Data/DATA.md`.
- **Milestones:** list the few checkpoints that matter.
