# 0-Project — reflection-agents

Concrete attempts only. Each attempt is a dated subfolder with its own `data/ docs/ outputs/ reports/ scripts/` and a `TASK_BOARD.md`. Attempts are append-only history — do not rewrite past attempts.

Lanes:

- active attempts: `0-Project/<date>_<name>/`;
- attempt-level notes and step handoffs: `0-Project/<attempt>/docs/<date>_<topic>.md`;
- future ideas not ready for execution: `0-Project/_parking-lot.md`;
- abandoned or superseded attempts: `0-Project/_archive/`, with stop reason and replacement if any.
