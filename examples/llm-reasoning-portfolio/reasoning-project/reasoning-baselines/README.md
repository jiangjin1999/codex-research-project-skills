# reasoning-baselines

reasoning-baselines is a formal project in the `reasoning` portfolio. It follows the shared single-project template so its map and the portfolio dashboard stay consistent.

## Navigate

- `PROJECT_BOARD.html` — this project's map (single page): a persistent **Back to dashboard** link, version status, a persistent overview, and five views (Progress, Docs, Tasks, Materials, References).
- `0-Project/` — concrete attempts (each a dated subfolder with its own data/docs/outputs/reports/scripts and a TASK_BOARD.md).
- `1-Docs/` — project-level discussions, design, methods, comparisons (`1-Docs/README.md` is the index).
- `2-Data/` — data notes only (`2-Data/DATA.md`).
- `3-Paper_Survey/` — literature and evidence (`evidence-matrix.md`).
- `4-Skills/` — project-local prompts/procedures (`4-Skills/PROJECT_SKILLS.md`).
- `_ai/` — AI working memory (project_overview, project_board_spec, task_plan, findings, progress).

Edit from this project root. Sync order: Markdown / `_ai` first, then `PROJECT_BOARD.html`, then the portfolio dashboard if a portfolio-level field changed.
